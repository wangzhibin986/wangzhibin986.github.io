Octopress搭建静态博客网站

By Wang Zhibin

发表于 2015-08-24

文章目录
  • 使用Octopress搭建静态网站的好处：
    • 命令行操作
    • 纯文本写博客
    • 定制性高
    • 纯静态
    • 版本化管理
    • 迁移成本低
    • 简洁的Ruby框架
    • Markdown语法 尤其适合程序员
  • 基本过程
    • 搭建环境
    • 纯文本写博客
    • 生成静态网页
    • 本地预览
    • 部署

环境准备

  • Git、Ruby、DevKit、MarkdownPad的安装与配置（简单，直接下一步，资源见连接课程资料）
  • 安装Octopress并设置默认主题
  • 资料与视频教程

各工具的作用

  • Git版本管理工具，将代码托管到GitHub上
  • Ruby+DevKit：生成静态网页
  • MarkdownPad:Windows下Markdown语法编辑器

安装Octopress

  • 从GitHub官网克隆Octopress至本地

    git clone git://github.com/imathis/octopress.git octopress
  • 安装依赖项

    gem install bundler

    bundle install

  • 安装并使用默认主题

    rake install

    生成博客与单页

  • 新建博客

    rake new_post[“title”]

  • 新建单页面

    rake new_page[danyemian]

    rake new_page[danyemian/page.html]

    用Markdown语法书写博客

    可以使用MarkdownPad、记事本、Vim、Emacs……等工具

    生成静态页面、本地预览

    rake generate
    rake preview
    http://localhost:4000

    部署网站至GitHub

  • 新建仓库

    username.github.io

  • 与本地Octopress目录绑定

    rake setup_github_pages

    rake deploy

    托管源码至GitHub

    git add .
    git commit -m ‘your message’
    git push origin source

    博客的自定义配置-完善主配置文件

    在_config.yml文件中，设置url、title、author等基本信息

    博客的自定义配置——使用第三方插件

  • 安装使用主题

    • 到GitHub上找到主题文件、按照提示克隆到本地
    • 安装主题（记得备份，否则原来的配置会被覆盖，所以最好在其他设置之前完成）

      rake install[‘themename’]
    • 重新生成
      rake generate
  • 增加评论功能(多说网)
  • 增加社会化分享（JiaThis）
  • 自定义404页面 (http://www.qq.com/404)
  • 自定义导航（自定义octopress\source_includes\custom\navigation.html）

自定义网站域名

  • 在GitHub上绑定自定义域名

    echo ‘wzb.link’ > source/CNAME
  • 解析域名至GitHub
    • 子域名（www.wzb.com）,创建CNAME记录指向charlie.github.io
    • 顶级域名（wzb.com）,使用A记录指向192.30.252.153（154）
