Maven是什么？

Maven是Apache软件基金会下的一个开源项目。Maven是基于项目对象模型(POM)，可以通过一小段描述信息来管理项目的构建，
报告和文档的软件项目管理工具。Maven 除了以程序构建能力为特色之外，还提供高级项目管理工具。由于 Maven 的缺省构建
规则有较高的可重用性，所以常常用两三行 Maven 构建脚本就可以构建简单的项目。由于 Maven 的面向项目的方法，许多
Apache Jakarta 项目发文时使用 Maven，而且公司项目采用 Maven 的比例在持续增长。（摘自百度百科）

Maven是如何运行和管理项目的

Maven是通过POM(Project Object Model)来实现项目的管理，完成系列任务。项目的属性、依赖、构建配置等信息都被抽象到项目对象模型中。

  • POM组成

    • 项目基本信息

      项目名称、官方网站、项目的开发者，贡献者列表、许可证列表等
    • 构建环境

      包含能在不同环境中激活的profile(开发环境，生产环境)，对不同的环境设置分开管理。
    • POM关系

      可以引入当前项目依赖的其他项目、定义父级项目和子项目、设置当前项目坐标等
    • 构建设置

      修改Maven的默认行为（修改源码和测试代码的位置），添加插件或把插件反应到项目的生命周期上、以及自定义生成参数等。

    所有的POM配置信息都反应在pom.xml里面，以节点（对象的方式）定义对项目各部分的管理

Meven中的插件与目标

Maven是一个插件的框架，不执行任何构建任务，所有任务都交给插件来完成。
  • Maven执行插件的语法

    pluginId:goalId

  • Maven中调用插件目标的两种方式
    • 将插件与生命周期绑定，执行生命周期
    • 直接执行插件目标

  • Maven中的常用插件与目标

    • maven-archetype-plugins(生成项目骨架)
    • maven-dependency-plugin(帮助分析项目依赖)
      • dependency:analyze分析项目依赖
      • dependency:tree分析项目依赖树
      • dependency:list列出项目所有已解析的依赖
    • maven-help-plugin
      • help:system打印Java环境变量和系统属性
      • help:effectivepom打印有效pom
      • help:effectivesettings打印有效的设置
      • help:describe描述Maven插件信息
    • maven-resources-plugin（提供清晰的项目结构，独立管理Java代码文件和资源文件，添加额外资源目录，对资源进行过滤）
    • maven-surefire-plugin（单元测试，跳过测试类可以配置这个插件）
    • jetty-maven-plugin（内置jetty容器，将Maven项目内置到容器中，自动更新项目到jetty容器中）
    • maven-enforcer-plugin（创建规则。。Java规则，maven规则等，团队开发时常使用）

      项目的生命周期阶段

    Maven的项目周期指项目的构建过程，它包含了一序列的有序阶段，而一个阶段就是构建过程中的一个步骤。
    一个生命周期可以绑定多个插件目标。指令 mvn package

Maven依赖管理

  • 依赖（构建）：项目运行时所需的第三方类库文件
  • Maven如何管理依赖：通过坐标管理依赖 POM中的dependencies
  • Maven支持传递性依赖，不必在POM中定义间接依赖，只需要定义直接依赖。Maven会隐式的把间接依赖的库加到项目中。
  • Maven依赖范围
    • compile(编译范围，默认范围，所有classPath中可用)
    • provided（已提供范围，不需要打包到War文件）
    • runtime(运行时范围)
    • test(测试范围)
    • system(系统范围)

Maven仓库

Maven仓库是一个存放了所有依赖的仓库，这个仓库通过依赖坐标对其进行管理。

  1. 本地仓库（默认仓库）

  2. 远程仓库（Maven中央仓库，pom.xml中可配置）

    3. 项目站点报告

    • 命令： mvn site mvn clean site
    • POM文件中配置description,url,organization，developers等信息，还可以用maven-site-plugin把站点设置为中文
